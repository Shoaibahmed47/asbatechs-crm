 CRM System Requirements for Office

We are developing an internal CRM system for our company to manage employees, leads, and departments efficiently. Below are the required features and solutions.

1. User Management

Admin should be able to create users (employees).

Each user will belong to a specific department/brand.

Admin should be able to view activity of each user.

Users should have login credentials (email + password).

2. Attendance System

The CRM must include a daily attendance tracking system.

Features required:

Clock In / Clock Out

Break Start / Break End

Track total working hours

Track total break time

Show attendance report per user per day

Admin Dashboard should show:

Who is currently active

Who is on break

Who is offline

3. Departments / Brands Management

Admin should be able to create and manage departments or brands.

Example departments:

ResumePro

Design

Book Publishing

Features:

Add / Edit / Delete Departments

Assign users to departments

Leads should be tagged by department

4. Manual Lead Entry (Hot Leads)

Users should be able to manually add HOT LEADS.

Fields required:

Client Name

Phone Number

Email

Source

Department

Notes

Assigned User

Lead Status

Example status:

New

Contacted

Follow Up

Closed

5. Manual Lead Entry (Sale Leads)

Users should also be able to add SALE LEADS manually.

Fields required:

Client Name

Phone

Email

Department / Brand

Sale Amount

Service Purchased

Assigned User

Notes

Date of Sale

Admin should be able to see sales reports per department and per user.

6. Website Lead Integration

If someone submits a form on our websites, the lead should automatically be saved in the CRM.

Required:

Form submissions from websites go directly to CRM

Leads should be tagged based on which website or service

Assigned to correct department

Example:

Resume website → ResumePro department

Design website → Design department

Publishing website → Book Publishing department

7. Lead Dashboard

CRM should have a lead dashboard showing:

Total Leads

Hot Leads

Sales Leads

Leads per department

Leads per user

Daily / weekly reports

8. Search & Filters

Users should be able to:

Search leads by name / phone / email

Filter by department

Filter by user

Filter by date

9. Notes & Activity Log

Each lead should have:

Notes section

History of actions

Which user updated the lead

10. Admin Control

Admin should be able to:

View all departments

View all users

View all leads

Export reports

✅ Goal of this CRM

Track employee attendance

Manage leads by department

Track sales performance

Collect leads from websites automatically

Improve team productivity
