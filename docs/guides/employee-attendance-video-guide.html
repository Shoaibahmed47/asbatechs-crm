Employee Attendance — Video Guide Script

Language: Simple Urdu / Hindi mix · Target length: ~5 minutes

Recording ke liye (simple)
1. Loom ya OBS se screen record karein.
2. Har scene ke neeche “Screen par dikhao” follow karein.
3. “Bolna” wala text naturally padh sakte hain — Urdu/Hindi mix OK hai.
4. Video khatam hone par file team ko WhatsApp / Drive / internal link se share karein.

Full video script (~5 min)

Scene 1 — Shuruat 0:00 – 0:35

Screen par dikhao

CRM login page ya Attendance page ka overview (sidebar + main area).

Bolna (script)

Assalam o Alaikum. Yeh Asbatech CRM ka Attendance system hai. Har roz aap yahan apna clock-in, break, aur clock-out record karte hain. Manager ko pata chalta hai aap kab aaye, kab break li, aur kitne ghante kaam kiya. Aaj main step-by-step dikhata / dikhati hun ke employee Attendance kaise use kare.
Scene 2 — Login aur Attendance page 0:35 – 1:10

Screen par dikhao

  1. /login par email + password enter karein → Login.
  2. Left sidebar mein Attendance click karein.
  3. Page par live status, buttons (Clock in / Start break / End break / Clock out) dikhao.

Bolna (script)

Pehle apna email aur password se login karein. Left side menu mein Attendance par click karein. Yeh page aapka daily attendance ka main page hai. Yahan aap apna status dekhte hain — working, break, ya shift end.
Scene 3 — Desktop Agent (ek baar laptop par) 1:10 – 2:15

Screen par dikhao

  1. Agent section: status badge (Not installed / Running / Stale).
  2. Install Agent → PowerShell command copy → Administrator PowerShell mein run.
  3. CRM password enter → install complete.
  4. Verify Agent click → badge Running ho.

Bolna (script)

Clock-in se pehle laptop par Desktop Agent install hona chahiye — yeh sirf ek baar karna hota hai. Install Agent button se command copy karein, PowerShell Administrator mode mein run karein, apna CRM password dein. Phir Verify Agent dabayein — jab Running dikhe, agent ready hai. Bina agent ke clock-in block ho sakta hai.
Scene 4 — Clock In (shift start) 2:15 – 2:55

Screen par dikhao

  1. Clock in button click.
  2. Status Active (working) ho jaye.
  3. Clock in time aur work timer start dikhao.
  4. Toast message (on-time / late feedback) agar aaye to dikhao.

Bolna (script)

Office aane ke baad Attendance page par Clock in dabayein. Status Active ho jayega aur aapka time count shuru ho jata hai. Monday se Friday working days hain — Saturday Sunday par clock-in band hai. Agar aap late aaye to system late minutes record karta hai; manager ko report mein dikhta hai.
Scene 5 — Break start (note zaroori) 2:55 – 3:50

Screen par dikhao

  1. Start break click → popup center par open.
  2. Break type select: Lunch, Prayer, Tea/coffee, Personal, Other.
  3. “Where are you going?” box mein likho (kam se kam 3 letters) — example: “tea kitchen mein”.
  4. Start break confirm → status Break.

Bolna (script)

Break ke liye pehle Start break dabayein. Popup mein break type choose karein aur likhna zaroori hai — aap kahan ja rahe hain, example lunch, prayer, tea. Kam se kam teen characters likhein. Start time automatic record hota hai. Manager ko break ki detail report mein milti hai.
Scene 6 — Break end (seedha button) 3:50 – 4:15

Screen par dikhao

  1. Wapas aane par End break click — koi second popup nahi.
  2. Status wapas Active.
  3. Break table mein start time, end time, duration dikhao.

Bolna (script)

Break khatam karne par seedha End break dabayein. Ab doosra popup nahi aata — end time aur total break duration automatic save hoti hai. Phir aap dubara working status mein aa jate hain.
Scene 7 — Zaroori rules (tab + laptop) 4:15 – 4:50

Screen par dikhao

  1. Attendance tab open rakho while clocked in.
  2. Tab close karke wapas aao → blue notice “Attendance tab reopened”.
  3. Optional: laptop sleep/lock notice dikhao.
  4. Batao: Gmail / YouTube other tabs OK — sirf Attendance tab close avoid karein.

Bolna (script)

Clock-in ke baad Attendance ka browser tab band mat karein — doosri sites ya Gmail, YouTube use kar sakte hain, lekin Attendance tab close hone par manager ko alert jata hai. Laptop lock ya sleep par bhi notice aa sakta hai — wake karke mouse move karein. Yeh rules fair attendance ke liye hain.
Scene 8 — Clock out + history + khatam 4:50 – 5:30

Screen par dikhao

  1. Shift end par Clock out.
  2. Today’s summary: work hours, break minutes.
  3. Date range / calendar se past days history scroll karo.
  4. Your punctuality card (week on-time days, streak) dikhao.

Bolna (script)

Kaam khatam par Clock out zaroor dabayein — is se aapke total hours finalize ho jate hain. Neeche aap apna summary aur purani dates bhi dekh sakte hain. Your punctuality card batata hai is week kitne din on-time aaye. Summary: Agent verify → Clock in → Break start with note → End break → Clock out. Koi sawal ho to manager ya admin se poochein. Shukriya!

Bonus scene — Late / Early / Absence popup (optional, +2 min)

Production mein yeh popup next working day par aata hai (weekend skip). Reason submit karna zaroori — bina iske clock-in band rehta hai.
SituationEmployee kya kare
Late arrival Next working day popup → reason likho (min 3 chars) → Submit → phir Clock in allowed
Early leave Shift se pehle clock-out → next day popup → reason submit
Absence (aaj nahi aaye) Jis din wapas aao → us date ka absence popup → reason submit

Quick checklist (employee handout)

  • ✅ Roz login → Attendance page
  • ✅ Agent Running (pehli baar install)
  • ✅ Clock in jab office start
  • ✅ Break: Start + note → wapas End break
  • ✅ Attendance tab open rakhein shift ke dauran
  • ✅ Clock out jab kaam khatam
  • ✅ Late / absence popup ignore mat karein — reason dena zaroori
